ICRC

Mops packages for ICRC

Mops is an onchain package manager for Motoko. Here are some Mops packages for ICRC tokens:

  • account: Implementation of ICRC-1 and ICRC-7 accounts.

  • account-identifier: Account identifiers for the ICP ledger.

  • auction: Package for auction functionality.

  • devefi-icrc-ledger: Local ICRC ledger syncing with remote ledger.

  • devefi-icrc-reader: Read the ICRC ledger transaction log.

  • devefi-icrc-sender: Send ledger transactions.

  • icrc1 and icrc1-mo: ICRC-1 libraries.

  • icrc1-types: ICRC-1 canister interface types.

  • icrc2-mo: ICRC-2 library.

  • icrc2-types: ICRC-2 canister interface types, including support for ICRC-1.

  • icrc2-batch: ICRC-2 batch transactions.

  • icrc3-mo: ICRC-3 library.

  • icrc30-mo: ICRC-30 library.

  • icrc4-mo: ICRC-4 library.

  • icrc45-mo: ICRC-45 library.

  • icrc7-mo: ICRC-7 library.

  • icrc75-mo: ICRC-75 library.

  • icrc-fungible: Support for fungible tokens ICRC-1, ICRC-2, ICRC-3, ICRC-4, and ICRC-10.

  • icrc-nft-mo: An ICRC NFT library.

  • origyn-nft: The reference implementation for the ORIGYN NFT.

  • ovs-fixed: ICRC-85 fixed open value sharing.

  • ovs-ledger: Open value sharing ledger.
